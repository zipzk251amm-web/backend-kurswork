# Інтернет-магазин електроніки

Курсова робота з дисципліни «Backend-розробка»

## Технології
- PHP 8
- MySQL
- MVC
- JSON (асинхронний обмін)
- HTML5, CSS3, JavaScript

## Встановлення
1. Скопіюйте папку `shop` в `C:\wamp64\www\`
2. Імпортуйте базу даних з файлу `database.sql`
3. Відкрийте `http://localhost/shop/`
